# OPPO Live Photo Viewer - v1.8 更新日志

**发布日期：** 2026-02-28

---

## 🎯 新功能

### 1. 鼠标滚轮切换图片 🖱️

- 在图片显示区域滚动鼠标滚轮可切换上/下一张图片
- 滚轮向上：上一张图片
- 滚轮向下：下一张图片
- 鼠标在左侧文件列表区域内时，滚轮用于滚动列表

### 2. 图片缩放算法优化 🖼️

- 缩小显示：使用 Bicubic 算法（更平滑）
- 放大显示：使用 Lanczos 算法（更锐利）
- 使用 Pillow 库实现高质量缩放

---

## 🐛 Bug 修复

### 1. 循环播放切换立即生效

- 修复切换循环播放开关后需要重新选择文件才能生效的问题
- 现在切换后自动重新加载当前图片

### 2. 主题切换 UI 恢复

- 修复深色/浅色主题切换后部分 UI 元素颜色未恢复的问题
- 完善 top_bar、bottom_bar、折叠按钮等元素的样式恢复

### 3. 文件列表重复显示

- 修复 `*.live.jpeg` 文件在列表中重复显示的问题
- 简化文件扫描 pattern，移除冗余的 `*.live.jpeg` 匹配
